# AI Personal Finance Guide

A simple **personal finance tracker** built with **HTML, Tailwind CSS, and Vanilla JS**.  
It helps users track income & expenses, calculate savings, and get basic AI-style financial suggestions.

## Features (Phase 1)
- Add income and expense transactions
- Display total income, total expenses, and savings
- Show basic AI suggestions based on spending habits

## Tech Stack
- HTML
- Tailwind CSS (via CDN)
- Vanilla JavaScript (ES Modules)

## Project Structure
```
Ai-personal-finance-tracker/
│── index.html
│── js/
│ ├── main.js

│ └── ai.js

```

## How to Run
1. Clone the repo
2. Open `index.html` in a browser
3. Start adding transactions!
